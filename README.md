## Simple flask api to fetch data from DIU server and serve it to the web app!

### How to use:

```
pip install -r requirements.txt
flask --app main run
```

#### Examples:

GET 127.0.0.1:5000/info?id={studentId}

GET 127.0.0.1:5000/result?id={studentId}&semester={semesterId}
